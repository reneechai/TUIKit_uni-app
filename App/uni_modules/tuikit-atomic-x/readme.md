# 腾讯云 · TUIKit uni-app 解决方案

<img src="https://qcloudimg.tencent-cloud.cn/raw/ec034fc6e4cf42cae579d32f5ab434a1.png" align="left" width=65 height=65> TUIKit 是腾讯云推出的一款适用于即时通信（Chat）和互动直播（Live）场景的 uni-app 解决方案。通过集成本方案，您可以快速为 App 添加聊天、通话、直播等功能。

<br clear="left"/>

## 产品特性

### Chat - 即时通信

<p align="center">
  <img src="https://web.sdk.qcloud.com/component/uni-app/assets/uniapp_chat.png"/>
</p>

- **单聊/群聊**：支持文本、图片、语音、视频、文件等多种消息类型
- **会话管理**：会话列表、消息记录、未读计数、消息搜索
- **联系人管理**：好友管理、黑名单、群组管理
- **消息能力**：消息撤回、转发、引用、@提醒、已读回执

### Live - 互动直播

<p align="center">
  <img src="https://web.sdk.qcloud.com/component/uni-app/assets/clipboard_20250903_054158.png"/>
</p>

- **完善的 UI 交互**：我们提供含 UI 的开源组件 TUILiveKit，可以节省您 90% 开发时间，您只需要花费30分钟就可以拥有一款类似抖音的在线直播应用。
- **多平台互联互通**：我们支持Android、iOS、uni-app 平台，您可以使用不同平台的 TUILiveKit 组件支持在线直播业务，未来我们还计划支持 Flutter、MacOS、Windows等平台。
- **互动弹幕**：我们支持自定义文本，自定义表情，房间信息提醒等多种互动信息展示。
- **互动礼物**：我们支持自定义礼物动画展示，费用自定义结算等功能。

## 开始使用

### Step 1：获取 SDKAppID 和密钥

前往 [腾讯云控制台](https://console.cloud.tencent.com/im) 创建应用，获取 SDKAppID 和密钥。

### Step 2：集成 TUIKit

参考接入文档将 TUIKit 集成到您的项目中：
- [Chat 接入文档](https://cloud.tencent.com/document/product/269/64507)
- [Live 接入文档](https://cloud.tencent.com/document/product/647/105442)

### Step 3：运行体验

配置好 SDKAppID 和密钥后，即可运行体验 Chat 和 Live 功能。
